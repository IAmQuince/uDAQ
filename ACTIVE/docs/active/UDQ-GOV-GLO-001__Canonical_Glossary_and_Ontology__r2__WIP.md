# Canonical Glossary and Ontology [SEC:UDQ-GOV-GLO-001::0]

## 1. Purpose [SEC:UDQ-GOV-GLO-001::1]

This glossary owns high-risk semantic terms used across the active UniversalDAQ corpus.

## 2. Additional control-tower terms [SEC:UDQ-GOV-GLO-001::2]

- **governance model** — the project-wide machine-readable graph of documents, requirements, worked examples, invariants, decisions, and intended implementation areas.
- **execution contract** — the implementation-facing derivative of the active governed corpus for the currently allowed slice.
- **invariant** — a future runtime conformance rule that guards structural, transitional, timing, or evidence truth.
- **coverage matrix** — the machine-readable and human-readable posture table showing which requirements are semantically closed, contract-defined, invariant-defined, deferred, or blocked.
- **decision log** — the controlled record of scope gates, deferred items, and explicit rulings.
- **executive summary** — the human-facing package state summary; informative but not a replacement for active controlled specs.

## 3. Existing high-risk operational terms [SEC:UDQ-GOV-GLO-001::3]

The active term-usage matrix remains the machine-readable companion to this glossary and shall continue to carry the detailed alias and confusability posture for terms such as requested state, applied state, observed state, restore, workspace state, machine state, live, historical, review, and live trace.
