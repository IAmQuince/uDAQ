# ui

UI-facing state models and pure view-model builders only. Widget-heavy implementation remains optional and deferred behind explicit Qt/pyqtgraph extras.

Current active scope includes:
- shell-facing state models and review posture summaries
- trusted-session and first-signal shell seams
- persistent top information bar summary
- dock/default-layout preferences with right-side control-dock default
- user-demo mode foundations and curated demo scenarios
- trace presentation, legend, and autosave preference models
- operator-shell foundation builders for future visible GUI embodiment
