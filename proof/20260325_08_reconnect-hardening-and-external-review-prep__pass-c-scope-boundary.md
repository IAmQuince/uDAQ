# 20260325_08 — Pass C Scope Boundary

## Purpose
Pass C is a final review-curation and document-classification-normalization pass for the external-review package.

## Allowed work
- tighten reviewer reading order
- tighten authoritative-versus-historical labeling
- normalize document-classification language for review surfaces
- add classification guidance, matrices, migration notes, and validators
- fix stale references, ambiguous wording, and navigation defects
- update debt-registry entries for intentional deferrals
- run integrity and review-readiness checks

## Not allowed
- new hardware behavior changes
- reconnect or startup code changes
- new exploratory testing
- broad spec rewrites beyond traceability and classification clarity
- architecture or product-scope widening

## Deliverable standard
Pass C must improve review clarity without changing the substantive claim line of the package.
