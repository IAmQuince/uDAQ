# ADR-0007 — Governance Compression and Delivery-First Rebalancing

Status: Accepted
Date: 2026-03-23

## Decision
For the current sprint line, reduce mandatory manual process steps for bounded changes while preserving automated gates and traceability tooling.

## Why
The package had become easier to govern than to contribute to. The critique correctly pushed toward a tighter link between process and delivered value.

## Consequence
Contributors now have a shorter quickstart path, and documentation updates are expected to be directly proportional to the semantic change made.
