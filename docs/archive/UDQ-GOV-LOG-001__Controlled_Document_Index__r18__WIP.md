# Controlled Document Index [SEC:UDQ-GOV-LOG-001::0]

## 1. Purpose [SEC:UDQ-GOV-LOG-001::1]

This index identifies the active controlled UniversalDAQ document set used for package assembly and audit review.

## 2. Active revision rule [SEC:UDQ-GOV-LOG-001::2]

The active working revision for a controlled document is the single revision present under `docs/active/`. Superseded revisions shall be preserved under `docs/archive/`.

## 3. Current active controlled set [SEC:UDQ-GOV-LOG-001::3]
This index is synchronized to `registries/active/universalDAQ_document_registry_r20.*` for the current package revision.
