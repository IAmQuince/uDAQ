# Controlled Document Update and Impact Process [SEC:UDQ-GOV-SOP-001::0]

## 1. Update order [SEC:UDQ-GOV-SOP-001::1]

For a governed conceptual change, the update order shall be:
1. governing specification or glossary
2. contradiction/duplication or decision registers as needed
3. requirement matrix and requirement registry
4. governance model and execution contract if affected
5. invariant registry if affected
6. executive summary / next actions / generated status outputs
7. rerun package audit

## 2. Rationale [SEC:UDQ-GOV-SOP-001::2]

This preserves a clear chain from human truth to machine-readable derivatives to package summaries.
